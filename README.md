# cordova-plugin-firebasex-inappmessaging

Firebase In-App Messaging module for `cordova-plugin-firebasex`.

This is an SDK-only plugin — it includes the Firebase In-App Messaging SDK but has no JavaScript API. In-App Messages are configured and triggered through the Firebase Console.

## Installation

```bash
cordova plugin add cordova-plugin-firebasex-inappmessaging
```

This plugin depends on `cordova-plugin-firebasex-core` which will be installed automatically.

## Usage

Once installed, Firebase In-App Messaging will automatically display messages configured in the Firebase Console when the app meets the trigger conditions.

No code changes are required in your app.
